# ASCII艺术转换器 - 忠实原色ASCII动画生成器

一个强大的Web应用程序，能够将图片和GIF文件转换为ASCII艺术动画，支持实时调节和动态预览。

## 🎯 项目特色

- **忠实原色**：完美还原原始图片的颜色信息
- **实时调节**：支持细节和速度的动态调整
- **动画支持**：完整的GIF动画ASCII化处理
- **键盘快捷键**：丰富的快捷操作支持
- **性能优化**：防抖技术和平滑过渡效果

## 🚀 核心功能

### 文件支持
- **图片格式**：PNG、JPG、BMP等静态图片
- **动画格式**：GIF动画文件
- **文件大小**：最大支持10MB
- **拖拽上传**：直观的文件拖放操作

### 细节控制 (2-15级)
- **防抖技术**：300ms延迟，避免频繁重新计算
- **实时预览**：调节后立即生成新的ASCII效果
- **性能优化**：智能防抖，减少计算资源消耗

### 速度控制 (1-10级)
- **实时生效**：速度调整立即应用到播放中
- **智能提示**：显示速度等级（慢/中/快）
- **平滑过渡**：无卡顿的动画切换

### 播放控制
- **播放/暂停**：一键控制动画状态
- **重置功能**：快速恢复初始设置
- **状态指示**：直观的按钮状态显示

### 键盘快捷键
| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停切换 |
| `R` | 重置到初始状态 |
| `F` | 增加播放速度 |
| `S` | 减少播放速度 |

## 📋 使用方法

### 基本操作流程

1. **选择文件**
   - 点击文件选择器选择图片/GIF
   - 或直接拖拽文件到页面

2. **自动处理**
   - 程序自动分析图像内容
   - 生成对应的ASCII动画

3. **实时调节**
   - 调节细节滑块改变字符密度
   - 调节速度滑块改变播放速度

4. **播放控制**
   - 使用播放/暂停按钮控制
   - 使用键盘快捷键快速操作

5. **状态重置**
   - 点击重置按钮恢复初始状态

### 细节调节技巧

- **低细节 (2-6级)**：适合快速预览和大图处理
- **中细节 (7-10级)**：平衡的显示效果
- **高细节 (11-15级)**：精细的字符表现，适合小图

### 速度调节建议

- **慢速 (1-3级)**：便于观察细节变化
- **中速 (4-7级)**：自然的动画效果
- **快速 (8-10级)**：紧凑的动态展示

## 💡 技术特性

### 性能优化
- **防抖技术**：细节调节时避免频繁重新计算
- **内存管理**：原始数据保存，支持动态重新生成
- **异步处理**：大文件处理不阻塞用户界面
- **错误恢复**：处理失败时自动回退到之前状态

### 视觉效果
- **原色保持**：智能颜色分析和映射
- **字符集优化**：基于内容选择最佳字符
- **过渡动画**：按钮交互的流畅反馈
- **进度提示**：处理过程的视觉状态

### 兼容性
- **现代浏览器**：支持Chrome、Firefox、Safari、Edge
- **移动设备**：响应式设计，支持平板和手机
- **ES6模块**：使用现代JavaScript特性

## 🔧 开发信息

### 技术栈

- **图像处理**：Canvas API、gifuct-js库
